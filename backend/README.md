# Centaur Brain Backend

Backend for the Centaur Brain Obsidian vault manager.

## Scripts

- `main.py`: FastAPI server for capturing and processing new content.
- `brain_cleaner.py`: Maintenance script for cleaning summaries, healing links, and refactoring concepts.
- `add_book.py`: Script to add book metadata to the vault.
- `rebuild_vault.py`: Full vault reconstruction from sources.
- `resume_rebuild.py`: Resume a partial vault rebuild.
- `reindex_covers.py`: Update cover images for all sources.
